# mac、技巧工具、软件、网站分享

## vscode在MacBook的开发技巧

##### 1.列模式
```
option + command + 上/下方向键
```
##### 2.整体移动
```
option + 上/下方向键
```

## MacBook的技巧

##### MacBook电脑显示已 .开头命名的文件 ，例如：.gitignore
```
command + shift + .
```

##### MacBook程序坞显示与隐藏
```
option + command + D
```

##### MacBook显示文件后缀名
```
1.顶部导航栏：访达
2.偏好设置
3.高级
```

## mac开启任何来源下载
