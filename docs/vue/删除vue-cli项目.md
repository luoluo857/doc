### 删除vue-cli项目

怎么删除vue-cli创建时保存的项目配置

我们一般用vue create 项目名,然后保存了创建的项目

![](https://gz-images.oss-cn-guangzhou.aliyuncs.com/vue.png)

现在我们需要把vue-cli、vue-hello、vue、iwen这几个之前手滑保存了的项目给删除，

我们只需要在自己的电脑上搜索出`.vuerc`这个文件,或者在C盘:Administrator这个文件夹下找到它，当然我的电脑是

![](https://gz-images.oss-cn-guangzhou.aliyuncs.com/vuerc.png)

然后把那几个json给删除就行了

删除后，如下：

![](https://gz-images.oss-cn-guangzhou.aliyuncs.com/vuerc1.png)

这样就可以变成最开始的样子了:
